📋 Usage Instructions

  1. Launch Application: Click the button above to open the converter
  2. Add PDF Files: Drag and drop or click to select PDF files (up to 50MB each)
  3. Configure Settings: Optionally adjust conversion settings via the settings panel
  4. Start Processing: Click "Start Processing" to begin conversion
  5. Monitor Progress: Watch real-time progress for each file
  6. Download Results: Download individual files or create a ZIP archive
⚠️ Browser Requirements: This application requires a modern browser with support for ES6, File API, and Web Workers. Recommended: Chrome 80+, Firefox 75+, Safari 13+, or Edge 80+.
